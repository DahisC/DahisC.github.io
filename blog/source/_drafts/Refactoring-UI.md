1. 根據功能設計畫面，而非版面
  - 只使用灰階－－專注以字型大小、間距以及對比凸顯重要訊息
  - 現階段先不需要考慮顏色
  - 摒除所有邊際情況
  - 對額外功能保持悲觀－－先產出最小可用版本

2. 網頁傳達的個性
  - 字型
    - 襯線 *Serif* - 專業、經典
    - 無襯線 *Sans Serif* - 有趣

  - 顏色 
    - 藍色 - 超級安全牌，誰不喜歡藍色？
    - 金色 - 高貴、典雅
    - 粉紅色 - 沒那麼嚴肅
    - 使用心理學挑選顏色不是最佳解，挑一個你覺得看起來很讚的顏色，然後問自己為什麼想選它

  - 圓角
    - 方正 - 嚴肅
    - 不那麼圓 - 沒特別想法
    - 很圓 - 有趣
