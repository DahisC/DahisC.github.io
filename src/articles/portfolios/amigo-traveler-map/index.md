# Amigo 旅行者地圖



## 介紹

__Amigo 旅行者地圖__是一個基於地圖提供服務的應用程式。

當時的發想是建立一個 APP，並且由程式主動蒐集一些藝文活動的資料，
但也接受使用者註冊為嚮導（發布活動）或是旅行者（查看活動）兩個角色，

## 作為組長的心路歷程

Amigo 旅行者地圖是 2021 年在職訓時的小組作業，每組有 5 人，當時的我則被指派為小組的隊長。

當時準備開發這個專案時，除了技術的選擇以外，比較需要克服的是小組中有些人才剛接觸程式，在對前、後端的概念或是寫法都還懵懵懂懂的情況，作為組長我應該怎麼帶領這個小組完成專案？
